Brewlore Studio

Brewlore Studio is a web-based content publishing tool for managing and uploading short-form educational videos about beverage history, whisky, Korean traditional alcohol, tequila, rum, gin, and related culture.

The tool uses TikTok Login Kit so the user can authorize access through OAuth. It uses TikTok Content Posting API to upload selected MP4 videos to the user's TikTok drafts or inbox for review before publishing.
